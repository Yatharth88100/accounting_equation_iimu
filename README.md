# Accounting Equation Tutor (No AI)

## What this project does
This project helps students practice Accounting Equation concepts by:
- Generating **new numerical questions**
- Giving **100% correct solutions**
- Using **fixed accounting rules**
- NOT using any AI or API

## Technologies Used
- Python
- Streamlit
- Rule-based logic

## Why answers are reliable
- Every question follows accounting equation rules
- No guessing or hallucination
- Same concept, different values every time

## How it works
1. Student selects a topic
2. App generates a new question
3. Accounting rules are applied
4. Solution is displayed with equation verification
